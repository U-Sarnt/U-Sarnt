## What I build / Qué construyo

**Security tooling**
Static analysis, agent security, CI/CD hardening. I care about findings that are deterministic, explainable, and bounded to repo-local evidence — not opaque scores.

**Network diagnostics**
CLI tools for host discovery, ARP scanning, traceroute, MAC vendor lookup, and OS fingerprinting. Built for engineers who want structured output they can actually pipe somewhere.

**Desktop automation**
GUI-first tools for access control and data cleanup workflows. Real applications used in real operational contexts, not demos.
